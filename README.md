# Mnemosyne

![build status](https://ci.peaterpita.com/api/badges/PeaterPita/Mnemosyne/status.svg)

A personal portfolio and blogging site where posts are stars collected in obsidian-like constellation graphs. Built using [Astro](https://astro.build), and deployed from my NixOS homelab.

**Live at: [peaterpita.com](https://peaterpita.com)**

---


## Deployment 
Deployment (and hosting) is handled through my own simple Woodpecker CI/CD workflow on my [Olympus](https://github.com/PeaterPita/Config.NixOS/blob/master/modules/system/servers/portfolio.nix) NixOS homelab.
